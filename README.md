# VueX博客主题系统
基于vue2和elementui开发的个人博客系统，可以登陆注册，首页展示，子菜单渲染，侧边工具栏等

# 本项目使用了本地mock数据
所以clone本项目之后，使用npm 或者 yarn 安装依赖包之后，就可以正常启动并展示效果
